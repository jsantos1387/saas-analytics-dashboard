# SaaS Analytics Dashboard

A SaaS analytics dashboard built with HTML, CSS, and JavaScript.

## Features

- Revenue dashboard metrics
- Customer management
- Add customers
- Edit customers
- Delete customers
- Search customers
- Filter by status
- Sort by revenue
- Dark mode
- CSV export
- Local storage persistence

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage API
